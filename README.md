# appShoppingList

## Aplicación web Backend y Frontend de lista de compras con sumatoria dinámica, solicitudes HTTP, agregado y eliminado de nuevos productos.
Tecnologías: NodeJS, Express, Angular, TypeScript, HTML, CSS y CORS.
